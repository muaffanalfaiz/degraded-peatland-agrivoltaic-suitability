# Data folder

This repository intentionally avoids uploading all raw data at the start.

## Recommended practice
Use this folder for:
- metadata
- tiny sample data if shareable
- public demonstration subsets

Do not upload:
- copyrighted or restricted source data
- very large raw rasters unless you deliberately want them public
- messy intermediate geodatabases

See `data/metadata/data_inventory_template.csv` for the inventory format.
