# Upload checklist

## Upload now
These files are safe and useful for the first public GitHub release:

- `README.md`
- `LICENSE`
- `CITATION.cff`
- `.gitignore`
- `docs/project_summary.md`
- `docs/methods.md`
- `docs/criteria_and_weights.md`
- `docs/data_sources.md`
- `docs/limitations.md`
- `docs/publication_status.md`
- `outputs/figures/*.jpg`
- `outputs/tables/ahp_weights.csv`
- `outputs/tables/suitability_area_stats.csv`
- `outputs/tables/peatland_mask_area_stats.csv`
- `results/key_findings.md`

## Upload later
- `arcgis-pro/project_clean.aprx`
- cleaned geoprocessing screenshots if you want
- sanitized toolboxes
- `scripts/*.py` once you are ready
- sample data / metadata inventory

## Do not upload right now
- the accepted manuscript DOCX
- publisher-like final PDF unless permitted
- messy local geodatabases
- raw datasets with unclear redistribution rights
- files with personal local paths or university account details

## Before pushing
Check these manually:
- remove personal Windows paths
- remove phone numbers if you do not want them public
- remove email addresses you do not want harvested
- confirm all figures are the public versions you want shown
